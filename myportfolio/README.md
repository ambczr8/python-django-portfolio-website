# Django Commands

* Running Server: `python3 manage.py runserver`
* Create Super User: `python3 manage.py createsuperuser`
* Execute Migration Scripts: `python3 manage.py migrate`
* Package Model Changes: `python3 manage.py makemigrations`
* Create Directory: `python3 manage.py startapp <app_name>`
* Create Starter Django Project Structure: `python3 -m django startproject <project_name`
* Install Django: `pip3 install django`
* Request static files: `python3 manage.py collectstatic`
